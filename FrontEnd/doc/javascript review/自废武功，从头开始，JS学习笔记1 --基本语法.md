**二、 字符串相关：**

**1、普通字符串：**

	//双引号
	"I'm a man!!!"
	//单引号	
	'I'm a man too!!!'


**2、转义字符：**

如果字符串内部既包含'又包含"怎么办？可以用转义字符\来标识，比如：
	
	//转移字符
	//    \'来代表一个字符
	console.log("I\'m \"OK\"");


**3、ASCII+Unicode字符：**

\x##形式的十六进制表示，例如：

	'\x41'; // 完全等同于 'A'

还可以用\u####表示一个Unicode字符：

	'\u4e2d\u6587'; // 完全等同于 '中文'
