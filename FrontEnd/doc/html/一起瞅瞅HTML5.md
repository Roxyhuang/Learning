**一、canvas元素用于在网页上绘制图形**

**(1).什么是canvas**

　　HTML5 的 canvas 元素使用 JavaScript 在网页上绘制图像。画布是一个矩形区域，您可以控制其每一像素。canvas 拥有多种绘制路径、矩形、圆形、字符以及添加图像的方法。

**(2).浏览器支持** 

IE9+, Firefox, Opera, Chrome,  Safari

1.`canvas对象表示一个HTML画布元素<canvas>，它没有定义自行的行
